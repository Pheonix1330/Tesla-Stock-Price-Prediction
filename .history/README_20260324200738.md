#  Tesla Stock Price Prediction using RNN & LSTM

##  Project Overview
This project focuses on predicting Tesla (TSLA) stock prices using Deep Learning techniques.  
It uses Recurrent Neural Networks (SimpleRNN) and Long Short-Term Memory (LSTM) models to learn patterns from historical stock data.

The model predicts future stock prices for:
- 1 Day
- 5 Days
- 10 Days

An interactive Streamlit web application is also developed for real-time predictions.

---

##  Dataset
- Tesla historical stock price dataset
- Features:
  - Date
  - Open
  - High
  - Low
  - Close
  - Adj Close
  - Volume

 Target Variable: **Closing Price**

---

##  Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
- Streamlit

---

##  Project Workflow

### 1. Data Preprocessing
- Handle missing values using forward fill (ffill)
- Feature selection (Close price)
- Data normalization using MinMaxScaler
- Create time-series sequences

### 2. Exploratory Data Analysis (EDA)
- Visualize stock trends
- Moving average analysis

### 3. Model Building
- SimpleRNN model
- LSTM model
- Dropout layers to prevent overfitting

### 4. Model Evaluation
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Actual vs Predicted comparison

### 5. Hyperparameter Tuning
- Tuned number of units and dropout rates
- Selected best model based on MSE

### 6. Future Prediction
- Multi-step forecasting (1, 5, 10 days)
- Recursive prediction method

### 7. Deployment
- Streamlit app for user interaction
- Predict future stock prices dynamically

---

##  Results
- LSTM outperformed SimpleRNN
- Best model achieved lowest MSE
- Accurate short-term predictions observed

---

##  Key Insights
- Stock prices follow sequential patterns
- LSTM captures long-term dependencies better
- Data scaling improves model performance
- Hyperparameter tuning enhances accuracy

---

##  Limitations
- Does not include external factors (news, economy)
- Stock market is highly volatile
- Limited to short-term predictions

##  Future Improvements
